## MovieVerse

MovieVerse is a movie and TV series recommendation web application that allows users to explore trending content, search titles, filter by categories, and save favorites, all powered by The Movie Database (TMDB) API.

This project was built as part of the ALX Frontend Engineering Program to demonstrate real-world frontend architecture, API integration, state management, and deployment.

## PROJECT OVERVIEW

MovieVerse provides users with an intuitive platform to:

- Discover trending movies and TV series

- Search for specific titles

- Filter content by country, genre, and year

- View detailed information about movies and series

- Save and manage favorite movies and TV shows

- The application focuses on performance, usability, and scalability, following modern frontend best practices.

## FEATURES & FUNCTIONALITY

- Trending Content

- Displays trending movies and TV series from TMDB

- Search Functionality

- Search for movies or TV series by title

- Advanced Filters

Filter by:

Country

Genre

Release year

Fully responsive design for mobile, tablet, and desktop screens
